QB/QBX INVENTORY INSTALLATION

Use this guide to register the `spraycloth`, `spraypaint`, `scene_paper`, and `sign_kit` items for qb-inventory or qbx_core setups.

---

1. REGISTER THE ITEMS

Open your framework item file.

Common location:
qb-core/shared/items.lua

For qbx_core, the common location is:
qbx_core/shared/items.lua

Add these entries inside the items table:

spraycloth = {
    name = 'spraycloth',
    label = 'Spray Cloth',
    weight = 200,
    type = 'item',
    image = 'spraycloth.png',
    unique = false,
    useable = false,
    shouldClose = true,
    description = 'A cloth used to wipe or clean spray paint.'
},

spraypaint = {
    name = 'spraypaint',
    label = 'Spray Paint',
    weight = 500,
    type = 'item',
    image = 'spraypaint.png',
    unique = false,
    useable = true,
    shouldClose = true,
    description = 'A spray paint can used to paint surfaces.'
},

scene_paper = {
    name = 'scene_paper',
    label = 'Scene Paper',
    weight = 100,
    type = 'item',
    image = 'spraycloth.png',
    unique = false,
    useable = true,
    shouldClose = true,
    description = 'Used to place a text scene.'
},

sign_kit = {
    name = 'sign_kit',
    label = 'Sign Kit',
    weight = 250,
    type = 'item',
    image = 'spraypaint.png',
    unique = false,
    useable = true,
    shouldClose = true,
    description = 'Used to place a text sign.'
},

---

2. INSTALL THE ITEM IMAGES

Open the image folder included with this resource:

peak-sprays/install/inventory_images

Copy these files:

spraycloth.png
spraypaint.png

Paste them into your inventory image directory.

Example:
qb-inventory/html/images/

---

3. RESTART RESOURCES

Restart qb-core or qbx_core, restart your inventory resource, or restart the server so the new item data and images are loaded.
